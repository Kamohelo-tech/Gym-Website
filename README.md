# Olympus Fitness: Commercial Club Management & Member Portal

## 📝 Description
A full-stack, responsive commercial web application built for fitness facilities. It features a modern user interface, member registration capabilities, class scheduling systems, and a public-facing portal designed to maximize business conversions.

## 🚀 Key Features
*   **Interactive Scheduling:** Clean UI displaying real-time class availability, times, and trainer assignments.
*   **User Management:** Secure user registration, profile updates, and membership tracking.
*   **Responsive Design:** Fully mobile-optimized layout using CSS Grid and Flexbox ensuring high performance across devices.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3, JavaScript (DOM Manipulation)
*   **Backend Support:** Node.js / Express (for routing and form submission handling)
